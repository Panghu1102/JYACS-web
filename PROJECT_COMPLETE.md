## 🎓 使用指南

### 快速开始（3 步）

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问管理界面
# 打开浏览器: http://localhost:3000/admin
```

### 构建网站

```bash
# 构建静态网站
npm run build

# 输出目录: JYACS-web-main/
```

### 部署网站

```bash
# 方法 1: 上传到服务器
# 将 JYACS-web-main/ 目录上传

# 方法 2: 使用 Vercel/Netlify
# 连接 GitHub 仓库并配置构建命令
```
